# Shark ecotourism and conservation behaviour — analysis code

Analysis code for Barnes *et al.* (202X), *Quasi-experimental evidence from Hawaii
demonstrates that shark ecotourism can positively influence shark conservation
behavior over time*.

The study tracks the shark-conservation behaviour of participants prior to (T1,
pre-tour) and two years after (T2) an immersive shark ecotourism programme in
Hawaii, USA, against a control group of members of the general public who did
not take the tour. The effect of tour participation is estimated with a
**modular, doubly-robust Bayesian estimator** combining inverse probability of
treatment weighting (IPW), inverse probability of attrition weighting (IPAW),
and g-formula standardisation of the outcome model.

## Data availability

The datasets analyzed in this study will be deposited and made freely accessible
in the Sydney eScholarship Repository upon acceptance of this article, and a link

## Repository contents

| File | Description |
|---|---|
| `Doubly_robust_tourism.ipynb` | Analysis notebook. Fits the joint propensity / retention model, the modular doubly-robust outcome model, diagnostics, the sensitivity analysis, and the E-value. |
| `idata_revised_dr.nc` | Posterior object — headline doubly-robust two-stage model. |
| `idata_revised_joint.nc` | Posterior object — fully-joint IPW × IPAW model (sensitivity check). |
| `tour_effect_dr.jpg` | Headline figure: posterior of the tour effect (doubly-robust model). |
| `tour_effect_revised.jpg` | Posterior of the tour effect (fully-joint model). |
| `evalue_posterior.jpg` | Posterior of the E-value (unmeasured-confounding sensitivity). |
| `overlap_diagnostics.jpg` | Propensity- and retention-score overlap (positivity diagnostics). |
| `smd_balance.jpg` | Pre- vs post-weighting standardised mean differences. |

The `.nc` files and `.jpg` figures are regenerated by running the notebook; the
small headline posterior (`idata_revised_dr.nc`) is included for convenience. The
larger `idata_revised_joint.nc` is git-ignored and rebuilt on a full run.

## Reproducing the analysis

Run `Doubly_robust_tourism.ipynb` top to bottom. It reads
`Shark_Ecotourism_Data.xlsx` (see Data availability) and writes the posterior
objects (`idata_revised_*.nc`) and the diagnostic figures.

### Environment

The analysis was developed in Python 3.11 with:

- `pymc` ≥ 5.9
- `arviz`
- `numpy`, `pandas`, `scipy`
- `matplotlib`, `seaborn`
- `openpyxl` (to read the `.xlsx` data file)

A pinned dependency list can be installed with `pip install -r requirements.txt`
(see that file for exact versions).

Claude code was used to ensure model revised code works and is annotated properly, as well as to initialize this repo.

## Method summary

- **Design.** Quasi-experimental repeated cross-section (respondents cannot be
  linked across waves because of the de-identification requirement).
- **Estimator.** Modular doubly-robust Bayesian estimator: a joint
  propensity-to-tour and retention-to-T2 model supplies fixed, stabilised
  (Hajek) weights, trimmed at the 1st/99th percentiles; a weighted Bayesian
  logistic outcome model is then fit on the T2 sample, and the marginal average
  treatment effect is recovered by the g-formula.
- **Sensitivity.** Results are reported across five weighting specifications, and
  an E-value bounds the influence of any unmeasured confounding.

See the notebook for full methodological detail and citations.
